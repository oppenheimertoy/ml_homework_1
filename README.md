# ДЗ1. Регрессия для предсказания цен автомобилей

## Что было сделано

### EDA
- Загружены данные
- Обработаны пропуски
- Удалены дубликаты по признакам
- Преобразованы типы данных
- Построены визуализации: pairplot, heatmap корреляций, scatter-plots

### Модели
1. LinearRegression на числовых признаках
2. LinearRegression со стандартизацией
3. Lasso с подбором alpha через GridSearchCV
4. ElasticNet с подбором alpha и l1_ratio
5. LinearRegression + OneHotEncoder
6. Ridge + OHE с подбором alpha - лучшая модель

### Результаты
| Модель | R^2 test | Business metric (примерно 10%) |
|--------|---------|------------------------|
| LR raw | ~0.60 | ~0.12 |
| Ridge + OHE | ~0.65 | ~0.15 |

### Что дало прирост
- Добавление категориальных признаков через OHE
- Подбор гиперпараметров через кросс-валидацию

### Что не удалось
- Столбец `name` удалён из-за большого числа уникальных категорий
- Столбец `torque` удалён из-за сложного формата,но он в целом и не особо нужен
